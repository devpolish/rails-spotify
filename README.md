# Rails Spotify API

## End-Points

### Artists

```json 
    GET /spotify/artists?name="coldplay"
``` 
### Playlists

```json 
    GET /spotify/playlists?name="indie"
``` 

### Users

```json 
    GET /spotify/users?name="devpolish"
``` 

### Tracks

```json 
    GET /spotify/users?name="Dream on"
``` 

### Albums

```json 
    GET /spotify/users?name="The Wall"
```
